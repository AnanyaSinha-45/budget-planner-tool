# Budget Planner & Forecasting Tool

A budget analysis and forecasting tool developed as an academic project. 
The system helps users track income and expenses, analyse spending patterns, 
and forecast future budget requirements using basic statistical methods.

## Features
- Income and expense tracking by category
- Monthly budget summary and report generation
- Simple forecasting using trend analysis
- Data flow documented via system diagrams

## Tech Stack
- Python (data processing and forecasting logic)
- CSV-based data input
- Regression concepts for forecasting

## Project Structure
- `budget_planner.py` — core logic for budget analysis and forecasting
- `sample_data/` — example CSV input file
- `docs/` — system documentation and data flow diagrams

## Concepts Applied
- Data flow diagram (DFD) design
- Budget analysis and variance reporting
- Forecasting using trend-based estimation
- System documentation and report generation

## Academic Context
Developed as part of coursework at the Institute of Engineering & Management (IEM), Kolkata.
B.Tech Computer Science & Business Systems — 2023–2027.
